# TickDone

[TickDone](https://app-for-me.vercel.app/) 是一个基于艾森豪威尔矩阵（四象限法）构建的任务管理应用，帮助用户根据“重要性”与“紧急性”对任务进行分类、跟踪进度，并可视化时间分布。通过直观的界面和简洁的交互方式，提升用户的专注力与时间管理效率。

**可以自定义**:
1. 任务面板标题
2. 任务进度条- 拖拽
3. 任务时间环 (颜色反映强度)
4. 专注计时
5. 可视化统计
